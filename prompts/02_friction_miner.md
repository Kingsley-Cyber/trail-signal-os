# Friction Miner Prompt

Find evidence that a repeated task fails or requires compensation.

For each observation capture the exact task episode, participant language, context, failure mode, workaround, source, date, independence group, confidence and limitations.

Search both direct and indirect language:

- annoying, hard to, keeps falling, leaks, tangles, wish it had;
- cut, sewed, taped, 3D printed, repurposed, homemade;
- with gloves, wet hands, one handed, in the dark, in wind;
- after use, cleaning, storage, vehicle transition, return reason.

Do not treat copied posts or syndicated videos as independent evidence. Include contradictions and cases where users accept the current workaround.
