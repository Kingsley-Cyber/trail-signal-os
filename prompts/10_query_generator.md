# Query Generator Prompt

Generate a balanced query pack for one activity and friction. Use `docs/12_query_grammar.md` and `data/search_query_templates.csv`.

Create queries for complaints, workarounds, field behavior, incumbent reviews, failures/returns, pricing, seasonality, community vocabulary and falsification. Include participant terms, aliases, regional terms, current year and prior comparable season where relevant.

Do not include only the proposed product name. The purpose is to discover whether the problem exists independently of the solution hypothesis.
