# Query Grammar

## Atomic query pattern

`[activity] + [task/friction phrase] + [context modifier] + [evidence marker]`

Examples:

- `bank fishing changing lures wet hands annoying`
- `trail running shorts restricted stride cut side seam`
- `dog walking gloves waste bag dispenser one handed`
- `family camping tent setup dark workaround`

## Evidence markers

### Complaint

`annoying`, `problem`, `hate`, `hard to`, `keeps falling`, `too bulky`, `wish it had`, `poor design`, `review`.

### Workaround

`DIY`, `modified`, `cut`, `sewed`, `taped`, `3D printed`, `homemade`, `hack`, `repurposed`, `how I carry`.

### Replacement and failure

`broke`, `leaked`, `failed`, `lost`, `returned`, `replacement`, `durability`, `after one season`.

### Context

`with gloves`, `wet hands`, `one handed`, `in the dark`, `in wind`, `while kneeling`, `with kids`, `with dog`, `inside kayak`, `after rain`.

## Query matrix

Generate combinations across:

- participant language and technical language;
- singular/plural and common aliases;
- current year and prior seasonal year;
- regional terms;
- direct product and product-absent behavioral wording;
- supportive and falsification queries.

Never rely only on queries containing the proposed product name; that creates confirmation bias.
